# BigQuery Schema for Album Dataset

```
+-------------------------------------+
|               Schema                |
+-------------------------------------+
| |- id: string (required)            |
| |- title: string (required)         |
| |- total_tracks: integer (required) |
| |- duration: integer (required)     |
| |- release_date: string (required)  |
| |- label: string (required)         |
| |- artist_ids: string (repeated)    |
+-------------------------------------+
```
